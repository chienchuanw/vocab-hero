# Lib

This directory contains utility libraries and helper functions for the Vocab Hero application.

## Structure

- `db/` - Database utilities and Prisma client
- `auth/` - Authentication utilities
- `srs/` - Spaced Repetition System (SM-2 algorithm)
- `tts/` - Text-to-speech utilities
- `utils/` - General utility functions

## Naming Conventions

- Utility files: kebab-case (e.g., `date-formatter.ts`, `srs-calculator.ts`)
- Test files: Same as utility with `.test.ts` suffix
