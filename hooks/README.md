# Hooks

This directory contains custom React hooks for the Vocab Hero application.

## Naming Conventions

- All hook files must start with `use` prefix (e.g., `useVocabulary.ts`)
- Use camelCase for hook names
- Test files: Same as hook with `.test.ts` suffix

## Examples

- `useVocabulary.ts` - Hook for vocabulary data management
- `useStudySession.ts` - Hook for study session state
- `useProgress.ts` - Hook for progress tracking
