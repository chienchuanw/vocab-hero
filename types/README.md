# Types

This directory contains TypeScript type definitions for the Vocab Hero application.

## Naming Conventions

- Type files: PascalCase with `.types.ts` suffix (e.g., `Vocabulary.types.ts`)
- Use `interface` for object shapes that may be extended
- Use `type` for unions, intersections, and utility types

## Examples

- `Vocabulary.types.ts` - Vocabulary item and group types
- `Study.types.ts` - Study session and quiz types
- `User.types.ts` - User and progress types
